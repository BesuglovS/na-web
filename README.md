# Портал Академии Наяновой

Главная страница-портал образовательной платформы **Академия Наянова** (nayanovaacademy.ru). Представляет собой каталог всех проектов экосистемы с карточками-ссылками на поддомены.

## Возможности

- **Каталог проектов** — единая точка входа во все образовательные проекты платформы
- **Адаптивная сетка** — карточки автоматически перестраиваются для мобильных и десктопных устройств
- **Индикаторы статуса** — бейджи «Активен» / «В разработке» для каждого проекта
- **Тёмная тема** — градиентный фон в стиле платформы
- **SEO-оптимизация** — Open Graph мета-теги для корректного отображения в соцсетях

## Запуск

Просто откройте `index.html` в браузере или запустите локальный сервер:

```bash
python -m http.server 8000
```

Затем откройте http://localhost:8000.

## Структура проекта

```
na-web/
├── index.html          # Главная страница-портал
├── NULogo.png          # Логотип Академии
├── favicon.ico         # Фавикон
├── deploy.ps1          # Скрипт деплоя
├── deploy-all.ps1      # Скрипт деплоя всех проектов
├── clone-repos.ps1     # Скрипт клонирования всех репозиториев
├── .env.example        # Пример переменных окружения
└── README.md           # Этот файл
```

## Деплой

```bash
# Деплой портала
./deploy.ps1

# Клонирование всех подпроектов
./clone-repos.ps1

# Деплой всех проектов
./deploy-all.ps1
```
